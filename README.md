# home-ssc-pages

这是一个用于归档和发布多个独立 HTML 页面的仓库。  
仓库中存放的页面内容彼此独立，不针对任何特定的域名、网站或服务。  
适合作为静态页面的集中存放与展示空间，便于浏览、分享与版本管理。

## 目录说明

```
.
├── index.html          # 仓库首页（可选）
├── pages/              # 存放各个独立 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── assets/             # 页面所需的静态资源（图片、样式、脚本等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/` — 每个独立的 HTML 页面文件，可单独访问。
- `assets/` — 页面共享或专用的资源文件，按类型归档。

## 页面归档说明

- 每个页面均为完整、自包含的 HTML 文档，尽量不依赖外部资源（除仓库内资源外）。
- 页面内容、用途、风格各异，彼此之间无强关联。
- 如需新增页面，请在 `pages/` 目录下创建新的 `.html` 文件，并确保资源引用路径正确。

## 维护说明

- 本仓库主要面向归档与展示，不提供运行环境或后端支持。
- 页面更新、删除或替换时，请保持目录结构清晰。
- 建议在每次更新后，检查页面资源引用是否有效。
- 欢迎提交 Issue 或 Pull Request 来改进页面内容或结构。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 目录下的任意 HTML 文件即可查看。
3. 也可通过 GitHub Pages 等服务部署后在线访问（需自行配置）。

## 许可

本仓库中的页面内容仅供归档与展示，不附带任何商业或推广意图。  
具体页面的版权与许可由其作者或原始来源决定。
